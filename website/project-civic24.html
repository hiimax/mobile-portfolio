The Challenge

Public service delivery in Lagos faces a critical hurdle: unreliable connectivity. Field agents often lose data while syncing reports in remote areas.

Simultaneously, the backend needed to handle burst traffic during emergencies (e.g., floods) without dropping a single request. The goal was 100% data integrity regardless of network state.

My Solution & Role

I designed an Offline-First architecture to decouple user actions from network availability.

1. Optimistic UI & Local Caching:
Attributes are written immediately to a local Hive database. The UI updates instantly, while a background job manager (WorkManager) queues the sync event.

2. Resilient Sync Engine:
The sync engine retries failed requests using exponential backoff. In case of conflict (e.g., two admins editing the same report), we implemented a Last-Write-Wins strategy based on server timestamps.

3. High-Performance geo-hashing:
To locate nearest agencies efficiently, I implemented Geohashing in MongoDB, reducing query time from 500ms to 12ms for radius searches.
